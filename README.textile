h1. MacVimrc

My vim dotfiles (.vimrc, .givmrc) I use "MacVim":http://code.google.com/p/macvim/ on Mac OS X, but most of the settings should work with Linux or Windows versions of Vim.

Some stuff in the config files is related to the plugins I have installed (most of them are specific to Ruby on Rails development).

* "rails.vim":https://github.com/tpope/vim-rails.git
* "Command-T":git://git.wincent.com/command-t.git
* "NERD Tree":https://github.com/scrooloose/nerdtree.git
* "NERD Commenter":https://github.com/scrooloose/nerdcommenter.git
* "dbext.vim":https://github.com/edsono/vim-dbext.git
* "cucumber.vim":https://github.com/tpope/vim-cucumber.git
* "fugitive.vim":https://github.com/tpope/vim-fugitive.git
* "surround.vim":https://github.com/tpope/vim-surround.git
* "TailMinusF":https://github.com/vim-scripts/TailMinusF.git
* "delimitMate":https://github.com/Raimondi/delimitMate.git
* "endwise.vim":https://github.com/tpope/vim-endwise.git
* "Textile":https://github.com/timcharper/textile.vim.git
* "Pastie":https://github.com/tpope/vim-pastie.git
* "vim-snipate":https://github.com/garbas/vim-snipmate.git
